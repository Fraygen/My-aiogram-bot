# 🤖 Telegram Bot: Gemini Pro Ultra

[![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)](https://python.org)
[![Aiogram](https://img.shields.io/badge/aiogram-3.25-green?logo=telegram)](https://docs.aiogram.dev)
[![OpenRouter](https://img.shields.io/badge/OpenRouter-API-orange)](https://openrouter.ai/)

Этот бот - мой личный мини-проект, созданный для изучения **aiogram** и работы с **LLM**. Бот умеет отвечать на вопросы, используя бесплатные модели через OpenRouter, конкретно в этом боте используется модель **gemma-3-4b-it:free**. Также бот умеет хранить и очищать (с помощью команды) историю диалога. Бот был написал за 1 день.

🔗 **Ссылка на бота:** @fraygenbot (https://t.me/fraygenbot)

---


### ✨ Возможности
*   🧠 Общение с ИИ.
*   💾 Запоминает контекст беседы (история до 10 последних сообщений).
*   🎛️ Удобное меню с кнопками.
*   📊 Логирование всех действий для отладки.
*   💰 Большие лимиты по сообщениям

### 🚀 Команды
*   `/start` - запуск и приветсвие бота
*   `/help` - руководство по использованию бота
*   `/clear` - очистка истории диалога (в памяти Gemma)
*   `/ask` - задать вопрос к Gemma

Также можно просто написать любое сообщение, а ИИ ответит
